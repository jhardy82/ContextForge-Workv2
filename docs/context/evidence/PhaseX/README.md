# Phase X — Trust-but-Verify Evidence

This folder documents where to find artifacts, logs, and transcripts for Phase X quality runs.

Locations (relative to repo root):

- logs/PhaseX/YYYYMMDD/phasex_{RunId}.jsonl — Pester TBV JSONL
- logs/PhaseX/YYYYMMDD/pssa_{RunId}.jsonl — PSSA TBV JSONL
- logs/PhaseX/YYYYMMDD/wu_{RunId}.jsonl — WU Harness TBV JSONL
- logs/PhaseX/YYYYMMDD/transcripts/ — Transcripts
  - pester_{RunId}.txt
  - pssa_{RunId}.txt
  - wu_{RunId}.txt
- build/artifacts/ — Runner artifacts (NUnit XML, console, metadata, verification)
- Outputs/WUScanSource/{RunId}/ — WU harness outputs and evidence

Verification reports:

- build/artifacts/Pester-Verification-{RunId}.json
- build/artifacts/PSSA-Verification-{RunId}.json
- build/artifacts/WUHarness-Verification-{RunId}.json

Notes:

- NUnit v2 (&lt;test-results&gt;) and v3 (&lt;test-run&gt;) headers are accepted.
- Console marker check is gated by context for Pester: "[PesterRun:{RunId}] START".
- Timestamps are validated within a small slack window; all artifacts are also hashed (SHA-256).
